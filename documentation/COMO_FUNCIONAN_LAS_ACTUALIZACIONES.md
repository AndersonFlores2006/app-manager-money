# Cómo Funcionan las Actualizaciones de Gestor Money

## ¿Qué son las actualizaciones automáticas?

A partir de la versión 1.1.0, Gestor Money incluye un sistema de actualizaciones automáticas que te permite mantener tu app siempre actualizada sin necesidad de usar Google Play Store. Las actualizaciones se distribuyen directamente desde GitHub Releases.

## ¿Cómo se verifican las actualizaciones?

### Verificación Automática
- La app verifica automáticamente cada 24 horas si hay nuevas versiones disponibles
- Si encuentra una actualización, te envía una notificación
- Esta verificación ocurre en segundo plano y no afecta el rendimiento de la app

### Verificación Manual
- Ve a **Configuración** → **Actualizaciones** → **Verificar actualizaciones**
- La app consultará inmediatamente si hay nuevas versiones

## ¿Cómo instalar una actualización?

1. Cuando hay una actualización disponible, verás un diálogo con:
   - La versión actual instalada
   - La nueva versión disponible
   - Lista de cambios y mejoras
   - Botón para descargar

2. Haz clic en **Descargar actualización**

3. La app descargará el APK desde GitHub:
   - Verás una barra de progreso
   - La descarga ocurre en segundo plano

4. Una vez descargado, se abrirá el instalador de Android

5. Confirma la instalación como cualquier otra app

## ¿Qué necesito para que funcione?

### Permisos Requeridos
- **Internet**: Para consultar GitHub y descargar actualizaciones
- **Almacenamiento**: Para guardar el APK descargado
- **Instalar aplicaciones**: Para instalar el APK actualizado

### Configuración en Android
En algunos dispositivos Android (especialmente 12+), necesitas permitir instalaciones desde fuentes desconocidas:

1. Ve a **Configuración** → **Aplicaciones** → **Gestor Money**
2. Activa **Instalar aplicaciones desconocidas**
3. O ve a **Configuración** → **Seguridad** → **Instalación de apps** → **Fuentes desconocidas**

## ¿Qué pasa si tengo problemas?

### No recibo notificaciones de actualización
- Reinicia la app para activar la verificación automática
- Verifica que tengas conexión a internet
- Algunos dispositivos desactivan tareas en background si la batería es baja

### No puedo instalar la actualización
- Verifica que tengas los permisos mencionados arriba
- Asegúrate de que el APK se descargó completamente
- En Android 12+, permite instalaciones desde fuentes desconocidas

### La app dice que estoy actualizado pero sé que hay nueva versión
- Espera 24 horas para la verificación automática
- O usa la verificación manual en Configuración

## ¿Es seguro?

Sí, las actualizaciones son completamente seguras:

- Los APK vienen firmados digitalmente con el mismo certificado que tu versión actual
- Se descargan directamente desde GitHub (servidor seguro)
- El proceso de instalación es el mismo que cualquier app oficial
- No se instala nada sin tu confirmación

## ¿Dónde puedo ver las versiones disponibles?

Todas las versiones están disponibles en:
**GitHub Releases**: https://github.com/AndersonFlores2006/app-manager-money/releases

Puedes descargar manualmente cualquier versión desde ahí si prefieres.

## ¿Puedo desactivar las actualizaciones automáticas?

Por ahora, las verificaciones automáticas están activas por defecto. Si necesitas desactivarlas temporalmente, puedes ignorar las notificaciones.

## Soporte

Si tienes problemas con las actualizaciones:
1. Verifica tu conexión a internet
2. Reinicia la app
3. Contacta soporte si el problema persiste

¡Mantente actualizado para disfrutar de las últimas mejoras y correcciones!
